# Market Insight Dashboard

Next.js 기반 투자 인텔리전스 시스템 대시보드

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 기능

- 포트폴리오 현황 조회
- 보유 종목별 수익률 표시
- 생각/메모 기록 및 검색
- 주식 가격 실시간 업데이트

## 기술 스택

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts (차트)
- Lucide React (아이콘)

## API 연동

백엔드 FastAPI 서버 (http://localhost:3000)와 연동하여 데이터를 표시합니다.
